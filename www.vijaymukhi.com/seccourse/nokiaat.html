We first install the nokia pc suite. We then run the hyper terminal program that comes standard with Windows. This program is found at menu programs, accessories, communications. After a small wait the program will ask us to key in the name of a connection, any name will do. We are then presented by a screen that asks for connection information, We click on the last drop down list box and choose our nokia pc suite connection, which in our case is named nokia 9300 USB. We click on ok and then cancel on the next dialog box. We then type at+cgmi and press enter. We see on the screen nokia but the characters we type are not echoed back on the screen. To remedy this we choose menu file and then properties, then click on button ascii setup and check the check box echo type characters locally. Click on ok twice. Now whatever characters we type are also echoed on the screen.

 

1) at+cgmi

 

Nokia

 

This gives us the name of the manufacturer in our case Nokia

 

2) at+cgmm

 

Nokia 9300

 

This at command gives us the model number of our phone
